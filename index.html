Welcome to the 1st IAG Python Bootcamp!

Presentation

The IAG Python Boot Camp is aimed to teach some Python skills to people that want to learn about this powerful computacional language. This will be a intensive course of two days for those that never had other opportunity before to try Python to those that have learned something about it. With lecture and hands-on session, all people will be able to learn and use Python.
We highlight that the examples shown will be in the field of Astrophysics. However, the event is open for non-Astrophysicists and the examples can be extrapolated to their respective fields.

This course is inspired in the previously Boot Camp from NASA and Berkely.

Motivation

It is known that Python is one of the most powerful tool to make science currently. This I Boot Camp intents to teach some skills about the most important tools offer by Python, especially for astronomers (and physicists)!

Come learn the basics of Python and everything you need to do astronomy using a modern and beautiful programming language!

Topics
